# SPECKit Plus — Short Notes

## What is SPECKit Plus?
SPECKit Plus : It is a framework that enable us to do spec driven development so effectively , it is and forked version of speckit with improved features release by panaversity such as PHR , ADRS , and we do spec driven development in it using slash command.

---

## Core Commands

### 1️⃣ `/constitution`
Using this command , we set consitution means principles of our project that AI must follow.

### 2️⃣ `/specify`
Using this command we create spec.md for our particular feature in which we write what and how we want to add particular feature and how'll it function.

### 3️⃣ `/plan`
Using this command Agent plans how to implement our feature.

### 4️⃣ `/tasks`
Using this command Agent create easy implementable tasks to achieve goal seamlessly.

### 5️⃣ `/implement`
Finally using this command agent starts implementing our tasks to achieve desired results.
